德州帳本（Firebase 多人即時同步版）

iPhone 發布方式：
1. 下載 ZIP 檔到「檔案」App。
2. 到 Netlify 登入。
3. 開啟 Sites / Deploy manually。
4. 上傳這個 ZIP 檔。
5. 等待完成後，Netlify 會給你一個網址。
6. 用 Safari 開啟網址。
7. 按 Safari 分享按鈕 → 加入主畫面。

使用方式：
- 每個人第一次輸入自己的名字。
- 同一群朋友輸入相同群組代碼。
- 所有人會即時看到買入、補碼與結算更新。
- 開新局後，舊資料仍會保留在月／年統計中。

重要：
- 不要刪除 Firebase 專案。
- 不要停用 Anonymous Authentication。
- 不要把 Firestore 規則改回 allow false。
