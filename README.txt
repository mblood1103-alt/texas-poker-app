德州帳本 v17（有牌局日期記號月曆版）

新增功能：
- 將 iPhone 系統日期選擇器改為帳本內建月曆。
- 有牌局紀錄的日期會顯示金色記號；同一天多局會顯示局數。
- 點擊有記號的日期，可立即查看當日紀錄與獎牌排名。
- 可切換上一個月、下一個月，並保留前一天、今天、後一天與全部紀錄。
- 保留指定日期獎牌排名、日／週／月／年排行榜、訪客觀看、查看紀錄、完成鎖定與修改牌局等功能。

更新方式：解壓縮後，將全部檔案覆蓋上傳 GitHub。


v18 更新：
- 累積統計同時保留盈虧排行榜與出勤排行榜。
- 出勤率以牌局數計算：玩家參加的已完成牌局數 ÷ 該期間全部已完成牌局數。
- 日、週、月、年都會各自重新計算出勤率。
- 盈虧排行榜同步顯示每位玩家的出勤局數與百分比。
- 出勤率相同時，依出勤局數、盈虧與名稱排序。
