德州帳本 v98

德州帳本 v17（有牌局日期記號月曆版）

新增功能：
- 將 iPhone 系統日期選擇器改為帳本內建月曆。
- 有牌局紀錄的日期會顯示金色記號；同一天多局會顯示局數。
- 點擊有記號的日期，可立即查看當日紀錄與獎牌排名。
- 可切換上一個月、下一個月，並保留前一天、今天、後一天與全部紀錄。
- 保留指定日期獎牌排名、日／週／月／年排行榜、訪客觀看、查看紀錄、完成鎖定與修改牌局等功能。

更新方式：解壓縮後，將全部檔案覆蓋上傳 GitHub。


v18 更新：
- 累積統計同時保留盈虧排行榜與出勤排行榜。
- 出勤率以牌局數計算：玩家參加的已完成牌局數 ÷ 該期間全部已完成牌局數。
- 日、週、月、年都會各自重新計算出勤率。
- 盈虧排行榜同步顯示每位玩家的出勤局數與百分比。
- 出勤率相同時，依出勤局數、盈虧與名稱排序。


【v19 新增：撲克牌型判定】
- 點選 5 張公牌與每位玩家 2 張手牌
- 可匯入目前牌局玩家，也可自行新增判定玩家
- 同一張牌不能重複選，已選牌會變灰
- 支援高牌、一對、兩對、三條、順子（含 A2345）、同花、葫蘆、鐵枝、同花順
- 正確比較所有踢腳牌，公牌最大時會判定平分
- 顯示每位玩家的牌型、最佳五張牌、獲勝或平分結果


【v20 調整：獨立牌型練習室】
- 移除「匯入本局玩家」，不再連動目前牌局與記帳資料。
- 牌型判定改為獨立練習介面，遇到爭議時可自行輸入公牌與手牌解答。
- 也可隨時新增練習玩家，自己點牌熟悉一對、兩對、三條、踢腳與平分規則。
- 原本記帳、歷史紀錄、盈虧排行榜與出勤排行榜均保留。


v21 更新：點任何公牌或手牌空格，會直接跳出選牌視窗；選完自動關閉，不必上下滑動尋找牌組。


v26 改良：本局驗帳顯示牌局日期；開新局從驗帳區移出，改為本局完成後獨立顯示的大按鈕。


v28 更新：
- 新建空白牌局後顯示「取消開新局」。
- 按下後刪除該空白新局並回到上一局。
- 一旦加入玩家或產生任何帳目資料，取消按鈕自動隱藏。


v29 修正：空白新局即使由舊版建立、缺少上一局標記，也會顯示取消開新局並正確回到上一局。


v36 更新：
- 查看紀錄的次數改為「今日查看」，每天 00:00 後重新從 1 次計算，不再累積以前天數。
- 每一位觀看者紀錄右側新增「刪除」按鈕，房主可單筆刪除。
- 保留原本「清空全部查看紀錄」功能。


v37 更新：查看紀錄顯示今日查看次數、總累積次數，並可展開查看每次觀看時間。


v38 更新：排行榜改為綜合排名，總盈虧 50%＋報酬率 50%，套用於本局、日／週／月／年及歷史排名；畫面顯示報酬率與綜合分數。

v39 更新：
- 玩家輸入最後拿回金額後按「儲存並完成」，該玩家卡片會自動收合並顯示「已結算」。
- 完成後畫面會自動滑到下一位尚未結算的玩家，減少多人結算時記錯人的情況。
- 已結算卡片可按「修改結算」重新展開修改。
- 若已結算玩家之後又新增、扣除、修改或刪除買入，會自動取消已結算狀態，避免金額變動後誤以為完成。


v82 更新：
- 排行榜名次圖示改為立體撲克籌碼。
- 第 1／2／3 名使用金／銀／銅籌碼，第 4 名起使用深色籌碼。
- 籌碼加入白色邊塊、雙層內圈、皇冠、黑桃與立體陰影，並同步套用本局排名、盈虧排行、買入排行、出勤排行與歷史排行。
